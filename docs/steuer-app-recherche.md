# Steuer-App Kanton St. Gallen – Recherche (Schritt 1)

## 1. Struktur der St. Galler Steuererklärung (Natürliche Personen)

### 1.1 Hauptformular – Steuererklärung (4 Seiten)

**Seite 1: Personalien & Familienverhältnisse**
- Name, Vorname, Geburtsdatum, Zivilstand (per 31.12.)
- Beruf, Arbeitgeber
- Konfession (relevant für Kirchensteuer)
- Register-Nr.

**Seite 2: Einkommen (Ziffern 1–9)**
- Ziff. 1: Einkünfte aus unselbständiger Haupterwerbstätigkeit (Lohnausweis)
- Ziff. 2: Einkünfte aus unselbständiger Nebenerwerbstätigkeit
- Ziff. 3: Einkünfte aus selbständiger Erwerbstätigkeit
- Ziff. 4: Wertschriften- und Guthabenerträge (Formular 3)
  - 4.1: Zinsen auf Bankguthaben
  - 4.2: Erträge aus Wertschriften
  - 4.3: Erträge aus Beteiligungen
- Ziff. 5: Einkünfte aus Liegenschaften (Eigenmietwert, Mieteinnahmen)
- Ziff. 6: Übrige Einkünfte (Sitzungsgelder, VR-Honorare, Lottogewinne etc.)
- Ziff. 7: Renten und Pensionen
- Ziff. 8: Einkünfte Quellensteuer
- Ziff. 9: Total Einkünfte

**Seite 3: Abzüge (Ziffern 10–24)**
- Ziff. 10: Berufskosten (→ Formular 4)
- Ziff. 11: Schuldzinsen
- Ziff. 12: Unterhaltsbeiträge / Alimente
- Ziff. 13: Beiträge AHV/IV/EO/ALV/NBUV (vom Lohnausweis)
- Ziff. 14: Beiträge berufliche Vorsorge (2. Säule BVG inkl. Einkauf)
- Ziff. 15: Beiträge Säule 3a
- Ziff. 16: Versicherungsprämien und Sparzinsen (→ Formular 6)
- Ziff. 17: Behinderungsbedingte Kosten
- Ziff. 18: Krankheits- und Unfallkosten (→ Formular 7)
- Ziff. 19: Liegenschaftsunterhalt (→ Formular 8)
- Ziff. 20: Zuwendungen/Spenden
- Ziff. 21: Weiterbildungskosten (max. CHF 12'000)
- Ziff. 22: Nettoeinkommen
- Ziff. 23: Sozialabzüge
  - 23.1: Kinderabzug (unter Schule/Ausbildung): CHF 7'500
  - 23.2: Kinderabzug (in Schule/Ausbildung): CHF 10'600
  - 23.3: Ausbildungskosten Kinder (→ Formular 10)
  - 23.4: Abzug für unterstützte Personen
- Ziff. 24: Steuerbares Einkommen

**Seite 4: Vermögen**
- Wertschriften und Guthaben (→ Formular 3, Wertschriftenverzeichnis)
- Liegenschaften
- Fahrzeuge, Lebensversicherungen (Rückkaufswert)
- Übrige Vermögenswerte
- Abzüglich Schulden
- Steuerbares Vermögen

## 3. Steuertarife Kanton St. Gallen 2024

### 3.1 Einkommenssteuer

- **Progressiver Tarif** bis max. 8.5% (einfache Steuer)
- **Vollsplitting** für gemeinsam Steuerpflichtige und Alleinerziehende
  - Steuersatz = Satz auf der Hälfte des Einkommens
- Einkommen bis CHF 10'000 = steuerfrei (Alleinstehende)
- Maximum: Über CHF 521'600 (gemeinsam) bzw. CHF 260'800 (alleinstehend) = 8.5%

### 3.2 Vermögenssteuer

- **Linear: 1.7 Promille** vom steuerbaren Vermögen
- Freibetrag: CHF 75'000 pro Person + CHF 20'000 pro minderjähriges Kind

### 3.3 Steuerfüsse (Multiplikatoren auf einfache Steuer)

| Ebene | Steuerfuss 2024 |
|-------|----------------|
| **Kanton** | 105% |
| **Gemeinde** | Variiert (61% Balgach – 162% Degersheim) |
| **Kirche** | 20–28% (je nach Konfession/Gemeinde) |
| **Gommiswald** | **103%** |

**Berechnung Gesamtsteuer:**
```
Einfache Steuer × (Kantonssteuer% + Gemeindesteuer% + Kirchensteuer%) / 100
+ Direkte Bundessteuer
= Gesamte Steuerbelastung
```

**Beispiel Gommiswald, reformiert (angenommener Kirchensteuerfuss ~23%):**
```
Einfache Steuer × (105 + 103 + 23) / 100 = Einfache Steuer × 2.31
```

### 3.4 Direkte Bundessteuer

- Separater progressiver Tarif (Bundesgesetz DBG)
- Max. 11.5% ab CHF 895'900 (Alleinstehende)
- Fahrkostenbegrenzung: Max. CHF 3'000

## 5. Technische Rahmenbedingungen

### 5.1 E-Tax SG / eFiling

- Ab Steuerperiode **2025** führt Kanton SG neue Webapplikation **E-Tax SG** ein
- Ersetzt die alte eTaxes-Desktop-Software
- Funktioniert auf allen Geräten (Computer, Tablet, Mobile)
- Vollständig digitale Einreichung möglich
- Login über E-Login/AGOV
- **Demo-Umgebung verfügbar:** etaxdemo.sg.ch

**Relevanz für App:** 
- Kein offizielles API von E-Tax SG bekannt
- Output der App sollte die Daten so strukturieren, dass der User sie in E-Tax SG übertragen kann
- Langfristig: Prüfen ob XML-Import in E-Tax SG möglich

### 5.2 Steuerkalkulator

- Offizieller Steuerkalkulator unter sg.ch verfügbar
- Berechnet Belastung pro Gemeinde
- **Kann als Validierungsreferenz dienen**

### 5.3 Steuerbuch

- Offizielle Sammlung aller Gesetzesartikel und Praxisfestlegungen
- Online verfügbar auf sg.ch
- Wichtige Referenz für Edge Cases und Auslegungsfragen

## 6. Datenschutz & API-Optionen

### 6.1 Anthropic API Datenschutz

| Option | Retention | Training | Eignung |
|--------|-----------|----------|---------|
| **Standard API** | 30 Tage (ab Sep 2025: 7 Tage) | Nein (Commercial Terms) | Akzeptabel mit Einwilligung |
| **Zero Data Retention (ZDR)** | Sofort gelöscht nach Verarbeitung | Nein | Ideal für Steuerdaten |
| **AWS Bedrock / Vertex AI** | Provider-Logging konfigurierbar | Nein | Alternative mit EU/CH-Hosting |

**ZDR-Details:**
- Nur für Enterprise/Team API Keys verfügbar
- Inputs/Outputs werden nach Verarbeitung sofort gelöscht
- Safety-Classifier-Ergebnisse werden behalten (Usage Policy)
- Files API ist von ZDR ausgenommen (Dateien bleiben bis manuell gelöscht)
- Erfordert Vereinbarung mit Anthropic Sales

### 6.2 Schweizer Datenschutzgesetz (nDSG)

**Relevante Anforderungen:**
- Steuerdaten = besonders schützenswerte Personendaten
- Informierte, explizite Einwilligung für Übermittlung ins Ausland (USA)
- Datenschutzerklärung mit Angabe aller Bearbeitungszwecke
- Recht auf Auskunft, Löschung, Datenportabilität
- Datenschutz-Folgenabschätzung (DSFA) wahrscheinlich nötig

**Mögliche Architektur-Entscheide:**
1. **API mit ZDR** – pragmatischste Lösung, erfordert klare Einwilligung
2. **AWS Bedrock (EU-Region)** – Daten bleiben in EU, näher an CH-Recht
3. **On-Premise/Self-Hosted** – Aktuell kein Claude-Modell für Self-Hosting verfügbar

### 6.3 Empfehlung

Für MVP: **Standard Anthropic API** mit klarer Datenschutzerklärung und Einwilligung. Die Daten werden gemäss Commercial Terms nicht für Training verwendet und nach max. 7 Tagen gelöscht. Für Produktiv-Release: ZDR-Vereinbarung oder AWS Bedrock prüfen.

## 8. MVP-Scope: Einfacher Steuerfall

### Zielgruppe für V1:
- Alleinstehend ODER verheiratet
- Unselbständig erwerbstätig (Lohnausweis)
- Kein Liegenschaftsbesitz
- Keine selbständige Erwerbstätigkeit
- Wohnsitz Kanton SG

### Benötigte Inputs:
1. Lohnausweis (PDF/Foto → Vision-Extraktion)
2. Bankkontoauszüge per 31.12. (Saldi, Zinsen)
3. 3a-Bescheinigung (falls vorhanden)
4. Krankenkassen-Jahresabrechnung
5. Personalangaben (Eingabeformular)

### Outputs:
1. Strukturierte Übersicht aller Einkünfte und Abzüge
2. Berechnetes steuerbares Einkommen und Vermögen
3. Geschätzte Steuerbelastung (Kanton + Gemeinde + Bund)
4. Optimierungsvorschläge ("Sie könnten CHF X sparen durch...")
5. PDF-Export oder druckbares Formular
